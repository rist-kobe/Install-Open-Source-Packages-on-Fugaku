## 概要 / Overview
このリポジトリは、オープンソースソフトウェア（OSS）を、スーパーコンピュータ「富岳」の利用者の環境に導入する手順を公開するためのものです。

This repository contains detailed instructions for installing open-source software packages on the supercomputer Fugaku.

公開している手順書には「富岳」固有のコマンドや設定が含まれます。これらの詳細については、[「富岳」利用者向けマニュアル](https://www.r-ccs.riken.jp/fugaku/user-manuals/)を参照してください。

Some documents include Fugaku-specific commands and settings. For details, please refer to [the Fugaku user manuals](https://www.r-ccs.riken.jp/en/fugaku/user-manuals/index.html).

**All resources are provided in Japanese only.**
